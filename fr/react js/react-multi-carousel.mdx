## Introduction

Dans cet article, vous apprendrez à intégrer un composant React Carousel à une application React.

## Qu'est-ce qu'un carrousel?

Un carrousel est un composant dynamique qui affiche une série d'images ou d'autres contenus de manière rotative, coulissante ou défilante. Il est souvent utilisé pour présenter des images de produits, des articles phares ou des témoignages sur des sites Web.

## Caractéristiques principales d'un carrousel :

- Diapositives multiples : il contient plusieurs diapositives ou panneaux, chacun affichant un élément de contenu différent.
- Commandes de navigation : les utilisateurs peuvent parcourir les diapositives à l'aide de boutons, de flèches ou de points.
- Avancement automatique : le carrousel peut passer automatiquement à la diapositive suivante après un intervalle spécifié.
- Conception réactive : il doit ajuster son apparence et son comportement en fonction de la taille de l'écran pour garantir une bonne expérience utilisateur sur différents appareils.

## Intégration d'un composant React Carousel à une application React JS

Il existe plusieurs packages de composants React Carousel qui peuvent être installés via [npm](https://www.npmjs.com/) pour ajouter le composant carrousel.
Aujourd'hui, nous allons utiliser [react-multi-carousel](https://www.npmjs.com/package/react-multi-carousel).

## Commencer
- Créer un nouveau projet React avec vite [En savoir plus sur la création de projets React avec vite](https://vite.dev/)

### Installer le package react-multi-carousel

```ts:Ouvrez le terminal et exécutez
npm i react-multi-carousel
```

### Configuration de travail minimale

Dans votre fichier `App.jsx`, importez le package react-multi-carousel et ajoutez le code de la configuration de travail minimale.

```ts:App.jsx
import { useRef } from 'react';
import Carousel from 'react-multi-carousel';
import 'react-multi-carousel/lib/styles.css';
import "./App.css";

//Add this code for a responsive carousel 
const responsive = {
  superLargeDesktop: {
    breakpoint: { max: 4000, min: 3000 },
    items: 5
  },
  desktop: {
    breakpoint: { max: 3000, min: 1024 },
    items: 3
  },
  tablet: {
    breakpoint: { max: 1024, min: 464 },
    items: 2
  },
  mobile: {
    breakpoint: { max: 464, min: 0 },
    items: 1
  }
};

function App() {

  return (
    <>
      <Carousel responsive={responsive}>
        <div style={{ backgroundColor: "red", width: 300, height: 300 }}>
          Item 1
        </div>
        <div style={{ backgroundColor: "purple", width: 300, height: 300 }}>
          Item 2
        </div>
        <div style={{ backgroundColor: "orange", width: 300, height: 300 }}>
          Item 3
        </div>
        <div style={{ backgroundColor: "aqua", width: 300, height: 300 }}>
          Item 4
        </div>
        <div style={{ backgroundColor: "violet", width: 300, height: 300 }}>
          Item 5
        </div>
      </Carousel>
    </>
  )
}

export default App
```

Dans votre `index.css`, ajoutez un style pour donner du remplissage au carrousel

```ts:index.css
body {
  padding: 100px;
}
```
